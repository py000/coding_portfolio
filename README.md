# Semester 3 Coding Portfolio

This repository contains my coding portfolio for Semester 3, covering various topics in computational social science.

## Structure

```
coding_portfolio/
├── topic_01/          # Topic 1: Web Scraping
│   ├── formative01_01/
│   └── formative_01_02/
├── topic_02/          # Topic 2: Natural Language Processing
│   ├── formative_02_01/
│   └── formative_0202/
├── codes/             # Additional code files
├── data/              # Data files
└── docs/              # Documentation and slides
```

## Topics Covered

### Topic 1: Web Scraping
- Formative exercises on web scraping techniques
- Data collection and extraction methods

### Topic 2: Natural Language Processing
- Text preprocessing and cleaning
- Sentiment analysis with VADER
- Topic modeling with LDA and BERTopic
- Machine learning for text classification
- LLM usage and supervised BERT models

## Requirements

- Python 3.10+
- Jupyter Notebook
- Required packages: pandas, numpy, scikit-learn, nltk, transformers, bertopic, vaderSentiment, gensim

## Usage

Each topic folder contains Jupyter notebooks with completed exercises. Navigate to the specific topic and formative assignment to view the code.
